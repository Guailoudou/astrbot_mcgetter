AstrBot 插件用于查询mc信息并使用图片展示

# 使用方法
/mchelp 
--查看帮助

/mc   
--查询保存的服务器

/mcadd 服务器名称 服务器地址 
--添加要查询的服务器

/mcdel 服务器名称 
--删除服务器
